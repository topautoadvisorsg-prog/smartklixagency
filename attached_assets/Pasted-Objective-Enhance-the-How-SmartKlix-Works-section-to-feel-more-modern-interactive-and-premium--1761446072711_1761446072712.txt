On hover:

⚡ Animations & Interactivity
1. Scroll Reveal (Entrance Animation)

Use GSAP ScrollTrigger or AOS (Animate on Scroll) to animate each step as it enters the viewport.

Sequence each step (Discover → Build → Automate → Grow) with a 100–150ms stagger delay.

Animation idea:

from: { opacity: 0, y: 40 },
to: { opacity: 1, y: 0, duration: 0.6, ease: "power2.out" }


The connecting line can draw itself (using an SVG line with a stroke-dashoffset animation).

2. Connecting Line Animation (Optional but premium)

Replace the static yellow line with an animated SVG path that draws as you scroll.

Example:

stroke-dasharray: 300;
stroke-dashoffset: 300;
animation: drawLine 2s ease forwards;
@keyframes drawLine {
  to { stroke-dashoffset: 0; }
}


Trigger this animation when the section scrolls into view.

3. Subtle Loop Animation (Optional)

Add a gentle pulsing or floating effect to each yellow circle icon:

@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-4px); }
}
.icon-circle {
  animation: float 4s ease-in-out infinite;
}


This gives a sense of “life” to the icons without being distracting.

4. Responsive Optimization

On mobile, stack steps vertically with a fade-in animation.

Disable or tone down the floating animation for performance.

💡 UX Enhancements

Add tooltips or subtle hover captions (e.g., “Learn about your audience” when hovering Discover).

Make each icon clickable and open a short modal/lightbox or scroll to a deeper explanation (optional future phase).

Keep color contrast strong and text consistent (don’t shrink mobile font too much).
