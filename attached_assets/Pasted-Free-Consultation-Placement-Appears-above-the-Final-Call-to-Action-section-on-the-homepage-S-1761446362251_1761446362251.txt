Free Consultation

Placement:
Appears above the Final Call to Action section on the homepage.

🏷️ Section Title

Limited-Time Offer: Free Consultation

💬 Subtitle

For a short time, SmartKlix is offering free one-on-one consultations.
We’ll review your website, automations, and digital systems — and show you how to scale faster.

🎯 CTA Button

Book My Free Consultation →
Links to /contact.html (no calendar integration — users submit availability manually).

🎨 Design Requirements
Element	Description
Layout	2-column (text left / animation right)
Background	Light gradient (#FFFFFF → #F9FAFB)
Text Color	Navy #0D1B2A
Button	Gold #F4B400 with soft glow hover
Typography	Poppins (headline), Inter (body)
Spacing	100px top / bottom
Visual	Animated network / particle field or floating sphere on the right
✨ Animation & Motion
Element	Animation
Section Entry	Fade-in from bottom + gentle rise
Headline/Subtext	Sequential slide-up (staggered 0.15s)
Button	Glow pulse on hover (box-shadow + brightness shift)
Right Visual	Continuous soft motion (particles / float / parallax)
Scroll Trigger	All animations triggered once via IntersectionObserver or AOS

Right-Side Visual Options:

Particle Field (preferred) — using tsparticles or lightweight vanilla JS:

Slow-moving gold/blue dots connecting/disconnecting

Subtle parallax shift on mouse move

Soft glow fade (opacity 0.6–0.9)

CSS Float Animation (fallback) — floating gold circle nodes w/ light pulse

🧱 Code Guidelines

No heavy frameworks (React or vanilla acceptable)

Keep animations CSS-based (use @keyframes + transform, avoid JS-heavy loops)

Semantic HTML5 structure

Use modular CSS (SCSS, Tailwind, or component-level)

Clean, readable code — no inline clutter
