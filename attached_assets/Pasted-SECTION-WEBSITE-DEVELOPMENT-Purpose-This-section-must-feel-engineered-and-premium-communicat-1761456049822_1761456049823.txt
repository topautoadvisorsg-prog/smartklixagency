⚙️ Developer Implementation Notes

Use pure CSS + IntersectionObserver for scroll effects — no GSAP or heavy JS.

Optimize mockup visuals for web (SVG or compressed PNG).

Parallax and light sweep should feel slow and elegant, never distracting.

Maintain all animations at ≤60fps — use will-change: transform, opacity.

Code must remain modular, semantic, and responsive (no hard-coded sizes).

Keep consistent margins, padding, and breakpoints with existing SmartKlix design system.

Reuse SmartKlix global CTA button style for uniformity.
