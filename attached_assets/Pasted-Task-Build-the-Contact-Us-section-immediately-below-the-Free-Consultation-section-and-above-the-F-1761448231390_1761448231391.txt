🧠 UX & Responsive Behavior

On desktop → two-column layout (form left, visual right).

On mobile/tablet → stack vertically, form first, animation below (centered).

Maintain at least 80–100px top/bottom padding for breathing space.

Ensure form validation messages are clean and subtle (no harsh red highlights).

Add ARIA labels and accessibility tags.

🧰 Developer Implementation Notes

Use lightweight CSS animations for floating and glow effects (avoid heavy JS).

Right-side animation can use:

Option A: tsparticles or lightweight canvas animation

Option B: simple CSS keyframes for pulse + float + glow.

Keep everything modular — wrap in <section id="contact-us"> block.

Code must remain fully responsive and optimized (no layout shift on scroll).
