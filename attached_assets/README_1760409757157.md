# Smart Klix Agency Website

A premium, professional website for Smart Klix - a real digital agency specializing in web design, automation, and business solutions. Built with clean, modular code that's easily editable and scalable for future client projects.

## 🌐 Live Website Structure

```
smartklix.com/
├── index.html              # Homepage with hero, services preview, team, FAQ
├── services.html           # Detailed service offerings
├── about.html              # Company story and full team profiles
├── contact.html            # Contact form and business information
├── legal/
│   ├── privacy.html        # Privacy policy
│   └── terms.html          # Terms of service
├── assets/
│   ├── css/style.css       # Complete stylesheet with CSS variables
│   ├── js/main.js          # Interactive functionality
│   └── images/
│       └── logo-light.png  # Smart Klix logo
└── README.md               # This documentation
```

## 🛠 How to Edit Content

### Updating Text Content
1. **Headlines & Descriptions**: Edit directly in HTML files
2. **Company Information**: Update in each page's contact sections
3. **Team Information**: Modify team member details in `about.html`
4. **Services**: Update descriptions and features in `services.html`

### Changing Colors
Edit CSS variables in `assets/css/style.css`:
```css
:root {
    --primary-navy: #0D1B2A;    /* Main brand color */
    --primary-gold: #F4B400;    /* Accent color */
    --text-secondary: #4F4F4F;  /* Body text */
    /* ... other variables */
}
```

### Updating Logo
1. Replace `assets/images/logo-light.png` with new logo
2. Ensure logo is optimized for web (PNG or SVG)
3. Maintain aspect ratio for 40px height display

### Adding New Pages
1. Create new HTML file following existing structure
2. Copy header/footer from existing pages
3. Update navigation links in all pages
4. Add page-specific CSS if needed

### Modifying Forms
**Contact Form Integration:**
- Form currently shows success message locally
- Ready for integration with services like:
  - Formcarry
  - Netlify Forms  
  - Custom backend endpoint

**To integrate:**
1. Update form action attribute
2. Add proper field names expected by service
3. Modify JavaScript success handling

## 🔧 Technical Features

### CSS Architecture
- **BEM Methodology**: Block__Element--Modifier naming
- **CSS Variables**: Centralized theming system
- **Mobile-First**: Responsive design from 320px up
- **Performance**: Optimized animations and transitions

### JavaScript Functionality
- **Mobile Navigation**: Hamburger menu with smooth transitions
- **FAQ Accordion**: Expandable Q&A sections
- **Sticky Header**: Header background changes on scroll
- **Sticky CTA**: Bottom call-to-action appears on scroll
- **Form Validation**: Client-side validation with error messages
- **Scroll Animations**: Elements fade in as they appear

### SEO Optimization
- **Semantic HTML5**: Proper heading hierarchy and structure
- **Meta Tags**: Title, description, and Open Graph tags
- **Schema Markup**: LocalBusiness and Service structured data
- **Fast Loading**: Optimized CSS and JavaScript
- **Mobile Responsive**: Google mobile-friendly design

### Browser Support
- **Modern Browsers**: Chrome, Firefox, Safari, Edge (last 2 versions)
- **Mobile Devices**: iOS Safari, Chrome Mobile
- **Accessibility**: ARIA labels and keyboard navigation
