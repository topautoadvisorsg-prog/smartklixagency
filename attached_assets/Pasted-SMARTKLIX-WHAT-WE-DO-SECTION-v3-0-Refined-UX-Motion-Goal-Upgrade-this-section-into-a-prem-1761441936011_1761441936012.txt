✨ Motion & Micro-Interactions
On Scroll:
Use IntersectionObserver to trigger:


Fade in + slide-up for each card (opacity 0 → 1, translateY(40px) → 0)


Stagger by 0.15s for cascading flow


Duration ~0.8s


Easing: cubic-bezier(0.25, 0.1, 0.25, 1)


On Hover:
Card lifts (translateY(-8px))


Shadow deepens to 0 10px 24px rgba(0,0,0,0.08)


Icon glows softly (CSS filter: drop-shadow(0 0 10px rgba(244,180,0,0.45)))


Cursor: pointer


Link Hover:
Arrow smoothly slides right (transform: translateX(6px))


Link text transitions to gold


Duration ~0.3s, same easing


Background Motion:
Add subtle parallax: the section background scrolls ~2% slower than cards.


Optional: very faint ambient gradient drift (CSS animation on background-position).
